# Requisitos No Funcionales - Evolut

## RNF-001
La aplicación será accesible desde un navegador web.

## RNF-002
La información será almacenada en PostgreSQL.

## RNF-003
Los usuarios deberán autenticarse para acceder a sus conversaciones.

## RNF-004
Cada usuario solo podrá acceder a su propia información.

## RNF-005
La interfaz deberá ser sencilla e intuitiva.

## RNF-006
Las conversaciones deberán conservarse entre sesiones.

## RNF-007
La aplicación utilizará una API de inteligencia artificial para generar respuestas.
