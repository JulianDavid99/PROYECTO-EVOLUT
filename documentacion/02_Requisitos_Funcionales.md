# Requisitos Funcionales - Evolut

## RF-001
El sistema permitirá registrar nuevos usuarios.

## RF-002
El sistema permitirá iniciar sesión mediante correo electrónico y contraseña.

## RF-003
El sistema permitirá cerrar sesión.

## RF-004
El sistema permitirá actualizar información básica del perfil.

## RF-005
El sistema permitirá crear una nueva conversación.

## RF-006
El sistema permitirá consultar el historial de conversaciones.

## RF-007
El sistema permitirá visualizar una conversación completa.

## RF-008
El sistema permitirá eliminar una conversación.

## RF-009
El usuario podrá enviar mensajes dentro de una conversación.

## RF-010
El sistema almacenará cada mensaje enviado por el usuario.

## RF-011
El sistema solicitará una respuesta al servicio de inteligencia artificial.

## RF-012
El sistema almacenará la respuesta generada por la inteligencia artificial.

## RF-013
El sistema mostrará las respuestas generadas dentro de la conversación.

## RF-014
Cada usuario solo podrá acceder a sus propias conversaciones.

## RF-015
Cada usuario solo podrá acceder a sus propios mensajes.

## RF-016
El sistema mostrará sugerencias iniciales de temas de reflexión al usuario.

## RF-017
El sistema permitirá iniciar una conversación a partir de una sugerencia o mediante texto libre.

## RF-018
El sistema permitirá asignar un título a cada conversación.

## RF-019
El sistema permitirá consultar conversaciones anteriores mediante el historial.
